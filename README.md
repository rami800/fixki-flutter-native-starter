# Flutter Native Starter Template

A simple Flutter Native starter template for Android and iOS with:
- 2 splash screens
- 1 home screen
- Material UI
- Placeholder-ready project metadata

## Supported placeholders
Before building, these are replaced automatically:
- `__APP_NAME__`
- `__PACKAGE_NAME__`
- `__IOS_BUNDLE_ID__`
- `__APP_NAME_SNAKE__`
- `__APP_DESCRIPTION__`

## Build
- `flutter pub get`
- `flutter run`
